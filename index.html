Investigator's Handbook

The Investigator's Handbook is a web application designed to accompany the popular tabletop board game "Elder Sign" by Fantasy Flight Games.


This app keeps track of various components -

  • Each player's sanity, stamina, and clues
  • Player number when assigning new investigators
  • Investigators that have been used already
  • The number of total Investigators

Investigator Card Anatomy -

Player Number. Investigator Name (Remove Investigator)
Sanity (Increase/Decrease Sanity by 1)
Stamina (Increase/Decrease Stamina by 1)
Clues (Increase/Decrease Clues by 1)
